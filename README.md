人工智能训练师（三级/高级）是上海市技能人才紧缺职业，上海市社保局等部门针对该职业推出了培训扶持、技能补贴、居住证积分等一系列政策，以下是结合上海社保局相关规定的详细信息整理：

1. **申报条件**：符合以下任一条件即可申报。包括从事本职业满10年；取得本职业或相关职业四级证书后满4年；取得大数据技术、计算机科学与技术等人工智能相关专业大专及以上学历并从事相关工作。

2. **培训开展**：上海多个区域已开展人工智能训练师（三级）培训班，例如徐汇漕河泾开发区、杨浦以及普陀桃浦镇未来岛产业园等地。培训课程依据国家职业技能标准设置，涵盖模型数据清洗与标注、智能数据分析等核心模块，采用“理论+实操”的模式。像首期漕河泾开发区的培训班就招收了50名园区内人工智能企业的技术研发骨干。

3. **证书相关福利**
    - **职业技能补贴**：市级补贴标准为3250元/人，在职人员可享受80%即2600元，上海户籍失业人员可领取全额3250元。此外，浦东户籍或缴纳浦东社保的人员、上海工会会员还可额外享受800元补贴。符合条件者可在证书颁发之日起12个月内，通过“上海人社”APP等渠道申请补贴。
    - **居住证积分**：该证书可用于上海居住证积分，能积60分，为非沪籍人员的居住和发展提供助力。

4. **评价与证书查询**：人工智能训练师（三级）已纳入上海职业技能等级认定，评价机构包括上海交通大学、上海市计算技术研究所有限公司等。其颁发的职业技能等级证书可在人社部技能人才评价工作网查询，具备较高的权威性和行业认可度。
